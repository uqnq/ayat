Lukas 12:49: Api yang Datang Membawa Perubahan Mendalam

"Aku datang untuk melemparkan api ke bumi, dan alangkah Aku menginginkan jika api itu sudah menyala!"

Ayat yang dicatat dalam Injil Lukas 12:49 ini adalah salah satu pernyataan Yesus yang paling kuat, sekaligus paling membingungkan bagi banyak orang. Diucapkan di tengah-tengah rentetan pengajaran mengenai kewaspadaan, kesetiaan, dan prioritas kerajaan Allah, frasa "Aku datang untuk melemparkan api ke bumi" bukanlah sebuah metafora ringan. Sebaliknya, ini adalah sebuah proklamasi radikal yang menantang pemahaman konvensional tentang misi Kristus. Saat banyak orang membayangkan Mesias sebagai pembawa kedamaian yang lembut dan penuh kasih, pernyataan ini memperkenalkan gambaran yang jauh lebih intens dan transformatif.

Pemahaman umum tentang Yesus seringkali terfokus pada pesan kasih, pengampunan, dan damai sejahtera. Namun, ayat ini, bersama dengan beberapa pernyataan lain dalam Injil, mengungkapkan sisi lain dari misi ilahi-Nya—sebuah misi yang juga melibatkan perpecahan, pertentangan, dan pemurnian yang seringkali menyakitkan. Api, dalam konteks Alkitab, adalah simbol yang kaya dan beragam, mewakili banyak hal mulai dari penghakiman ilahi hingga kehadiran Roh Kudus yang memurnikan. Oleh karena itu, untuk menggali makna Lukas 12:49 secara komprehensif, kita harus menyelam ke dalam kedalaman simbolisme api dan implikasi teologis dari kedatangan Yesus dengan membawa "api" ini ke dunia.

I. Makna Simbolis "Api" dalam Konteks Alkitab

Untuk memahami sepenuhnya pernyataan Yesus ini, kita perlu memahami kekayaan simbolisme api dalam tradisi Yahudi dan Kristen. Api bukanlah sekadar elemen destruktif, melainkan sebuah kekuatan multifaset yang merangkum berbagai aspek ilahi dan manusiawi.

1. Api sebagai Lambang Kehadiran Ilahi dan Kekudusan

Dalam Perjanjian Lama, api seringkali muncul sebagai penanda kehadiran Allah yang kudus dan mahakuasa. Semak duri yang menyala di hadapan Musa (Keluaran 3:2) adalah contoh ikonik. Api itu membakar tetapi tidak menghabiskan semak, menunjukkan kekudusan dan kekuatan Allah yang transenden. Demikian pula, tiang api yang memimpin bangsa Israel di padang gurun (Keluaran 13:21) melambangkan bimbingan dan perlindungan ilahi. Kehadiran api di Gunung Sinai (Keluaran 19:18) menandakan kemuliaan dan kekudusan Yahweh yang luar biasa. Konteks ini menegaskan bahwa ketika Yesus berbicara tentang "api," Dia mungkin merujuk pada membawa kehadiran Allah yang intens dan transformatif ke bumi.

2. Api sebagai Agen Pemurnian dan Penyucian

Salah satu makna terpenting api adalah kemampuannya untuk memurnikan. Dalam proses metalurgi kuno, api digunakan untuk memisahkan logam mulia dari kotorannya (Maleakhi 3:2-3). Dengan cara yang sama, api ilahi sering digambarkan sebagai alat untuk memurnikan umat Allah dari dosa dan kenajisan. Melalui penderitaan dan ujian, umat diuji seperti emas yang dimurnikan dalam api. Konsep ini sangat relevan dengan misi Yesus; kedatangan-Nya bertujuan untuk membersihkan dan menyucikan hati manusia, komunitas iman, dan bahkan dunia itu sendiri dari segala sesuatu yang tidak sesuai dengan kehendak Allah. Pemurnian ini seringkali bukan proses yang nyaman, melainkan sesuatu yang membakar dan menuntut perubahan radikal.

3. Api sebagai Simbol Penghakiman dan Kerusakan

Tidak dapat dipungkiri, api juga merupakan simbol penghakiman ilahi dan kehancuran. Api dan belerang yang menghancurkan Sodom dan Gomora (Kejadian 19:24) adalah peringatan yang mengerikan tentang konsekuensi dosa. Nubuat-nubuat para nabi seringkali menggunakan api sebagai gambaran murka Allah yang akan datang atas ketidaktaatan dan ketidakadilan. Dalam eskatologi (ajaran tentang akhir zaman), api sering dikaitkan dengan penghakiman terakhir dan pemusnahan kejahatan. Jika Yesus membawa api penghakiman, itu berarti kedatangan-Nya akan memisahkan terang dari gelap, kebenaran dari kepalsuan, dan menghasilkan konsekuensi yang jelas bagi mereka yang menolak panggilan-Nya.

4. Api sebagai Semangat, Gairah, dan Antusiasme

Di sisi lain, api juga dapat melambangkan semangat yang membara, gairah yang mendalam, dan antusiasme yang tak tergoyahkan. Dalam Kisah Para Rasul 2, lidah-lidah api muncul di atas kepala para rasul pada hari Pentakosta, menandai pencurahan Roh Kudus yang memberi mereka keberanian dan kuasa untuk memberitakan Injil. Api ini adalah api yang menginspirasi, memberdayakan, dan mengobarkan hasrat untuk melayani Allah. Konteks ini menunjukkan bahwa "api" yang Yesus maksud bisa jadi adalah semangat yang membara bagi Kerajaan Allah, yang mendorong orang percaya untuk secara aktif menyebarkan pesan-Nya dan hidup sesuai dengan nilai-nilai-Nya.

II. Misi Yesus: Bukan Damai Sejahtera, Melainkan Api

Pernyataan "Aku datang untuk melemparkan api ke bumi" menjadi semakin menarik ketika kita mempertimbangkan konteks yang lebih luas dari ajaran Yesus, terutama ayat-ayat selanjutnya dalam Lukas 12:

"Apakah kamu menyangka, bahwa Aku datang untuk membawa damai di atas bumi? Bukan, kata-Ku kepadamu, bukan damai, melainkan pertentangan. Karena mulai dari sekarang akan ada lima orang dalam satu rumah tangga yang terbagi-bagi, tiga melawan dua dan dua melawan tiga. Mereka akan saling bertentangan, ayah melawan anaknya laki-laki dan anak laki-laki melawan ayahnya, ibu melawan anaknya perempuan dan anak perempuan melawan ibunya, ibu mertua melawan menantunya perempuan dan menantu perempuan melawan ibu mertuanya." (Lukas 12:51-53)

Ayat-ayat ini menjelaskan dampak langsung dari api yang dibawa Yesus. Ini bukan api yang hanya membakar secara fisik, tetapi api yang menyalakan perpecahan dan pertentangan. Mengapa Yesus, yang disebut "Raja Damai" (Yesaya 9:5), menyatakan bahwa Dia datang bukan untuk membawa damai, melainkan pertentangan?

1. Api yang Mengungkap Kebenaran dan Memisahkan

Kedatangan Yesus dengan kebenaran-Nya yang radikal akan selalu menjadi titik perpisahan. Seperti api yang memisahkan logam murni dari ampasnya, ajaran Yesus memaksa orang untuk membuat pilihan yang jelas. Ini bukan tentang memecah belah demi perpecahan itu sendiri, tetapi untuk mengungkapkan apa yang tersembunyi dalam hati manusia. Apakah seseorang akan menerima Kerajaan Allah atau menolaknya? Apakah mereka akan mengikut Yesus atau tetap pada cara dunia? Pilihan ini seringkali memecah belah keluarga, teman, dan masyarakat, karena beberapa akan merangkul terang sementara yang lain memilih untuk tetap dalam kegelapan.

2. Api yang Mempertanyakan Status Quo

Yesus datang untuk menantang kemapanan agama, sosial, dan politik pada zamannya. Dia mengkritik kemunafikan para pemimpin agama, membalikkan meja para penukar uang di Bait Allah, dan bergaul dengan orang-orang yang dianggap rendahan oleh masyarakat. Ajaran-Nya tentang kasih, keadilan, dan belas kasihan adalah revolusioner dan seringkali mengancam tatanan yang sudah ada. Api yang dibawa-Nya adalah api yang membakar norma-norma lama, tradisi-tradisi yang kosong, dan sistem-sistem yang menindas. Ini adalah api yang memprovokasi pemikiran ulang, perubahan, dan seringkali penolakan dari mereka yang diuntungkan dari status quo.

3. Konflik Batin dan Eksternal

Api perpecahan ini bekerja dalam dua dimensi: internal dan eksternal. Secara internal, ajaran Yesus memicu konflik dalam diri individu. Setiap orang yang mendengar dan merespons panggilan-Nya akan mengalami pergolakan batin antara keinginan daging dan tuntutan roh, antara kenyamanan dunia dan panggilan pengorbanan. Ini adalah api yang membakar keraguan, dosa, dan kompromi dalam hati. Secara eksternal, konflik ini terwujud dalam pertentangan sosial. Mereka yang mengikuti Yesus akan menemukan diri mereka dihadapkan pada penolakan, ejekan, atau bahkan penganiayaan dari mereka yang tidak mengerti atau tidak setuju dengan jalan mereka.

III. Api Roh Kudus: Katalis Transformasi

Ketika Yesus menyatakan, "alangkah Aku menginginkan jika api itu sudah menyala!", ini menunjukkan keinginan yang mendalam dan mendesak. Mengingat janji-Nya tentang Roh Kudus, sangat mungkin bahwa "api" ini juga secara kuat merujuk pada Roh Kudus itu sendiri, yang akan dicurahkan setelah kenaikan-Nya.

1. Hari Pentakosta: Manifestasi Api Roh Kudus

Peristiwa Pentakosta, yang dicatat dalam Kisah Para Rasul 2, adalah penggenapan yang jelas dari "api" yang Yesus inginkan. Lidah-lidah api yang hinggap di atas kepala para murid adalah manifestasi fisik dari pencurahan Roh Kudus. Api ini tidak menghancurkan, tetapi memberdayakan. Para murid, yang sebelumnya takut dan bimbang, tiba-tiba dipenuhi keberanian, kemampuan untuk berbicara dalam berbagai bahasa, dan semangat yang membara untuk memberitakan Injil. Ini adalah api yang menginspirasi misi global Gereja, memicu pertumbuhan yang eksplosif, dan mengubah dunia.

2. Fungsi Roh Kudus sebagai Api

  • Memurnikan: Roh Kudus bekerja untuk menguduskan orang percaya, membakar habis dosa, egoisme, dan ketidakmurnian dalam hati kita. Ini adalah proses pembentukan karakter yang terus-menerus, seringkali melalui "api" ujian dan disiplin.
  • Menerangi: Roh Kudus adalah Roh kebenaran yang menerangi pikiran kita untuk memahami firman Tuhan dan kehendak-Nya. Seperti api yang memberi terang, Roh Kudus menghilangkan kegelapan ketidaktahuan dan kebingungan.
  • Menghangatkan dan Mengobarkan: Roh Kudus membangkitkan kasih, gairah, dan semangat dalam diri orang percaya. Ini adalah api yang membuat hati kita "membara" dengan kasih untuk Allah dan sesama, mendorong kita untuk melayani dengan antusiasme yang tak tergoyahkan.
  • Memberi Kuasa: Api Roh Kudus adalah sumber kekuatan ilahi. Dia memberdayakan orang percaya untuk menjadi saksi Kristus, untuk melakukan pekerjaan-Nya, dan untuk mengatasi tantangan dunia. Tanpa api ini, pelayanan kita akan hampa dan tanpa daya.

3. Keinginan Yesus agar Api Itu Menyala

Kerinduan Yesus agar api itu "sudah menyala" menunjukkan urgensi dan intensitas misi-Nya. Dia tahu bahwa api ini adalah satu-satunya cara untuk membawa pembaruan yang sejati dan transformatif ke bumi. Api ini akan membersihkan, memisahkan, dan mengobarkan, menciptakan komunitas orang percaya yang murni, bersemangat, dan penuh kuasa. Keinginan-Nya ini adalah panggilan bagi kita, sebagai pengikut-Nya, untuk juga merindukan dan mengupayakan agar api Roh Kudus terus menyala dalam hidup kita dan melalui kita ke dunia.

IV. Dampak Api Kristus dalam Sejarah dan Kehidupan Pribadi

Api yang dibawa Yesus bukanlah peristiwa sesaat; dampaknya terus berlanjut sepanjang sejarah dan masih terasa dalam kehidupan individu hari ini. Ini adalah api yang terus-menerus memicu perubahan dan transformasi.

1. Api dalam Sejarah Gereja

Sejak Pentakosta, sejarah Gereja telah ditandai oleh kobaran api ini. Kebangunan rohani, gerakan misi global, reformasi, dan bahkan periode penganiayaan semuanya dapat dilihat sebagai manifestasi api Kristus. Dalam setiap periode, api ini telah memurnikan Gereja, membangkitkan semangat baru, dan memisahkan gandum dari sekam. Para martir yang berani menghadapi kematian demi iman mereka adalah contoh nyata bagaimana api ilahi mengobarkan keberanian dan kesetiaan yang tak tergoyahkan.

Gerakan-gerakan reformasi, seperti Reformasi Protestan pada abad ke-16, juga dapat dilihat sebagai api yang membakar tradisi-tradisi yang menyimpang dan mengembalikan Gereja pada kebenaran Alkitab. Ini adalah proses yang penuh pergolakan dan seringkali berujung pada perpecahan institusional, tetapi pada intinya adalah upaya untuk memurnikan dan mengembalikan integritas iman. Api ini terus menyala dalam setiap kebangunan rohani, ketika Roh Kudus bekerja secara luar biasa untuk memperbarui dan menggerakkan umat-Nya menuju komitmen yang lebih dalam.

2. Api dalam Kehidupan Pribadi Orang Percaya

Secara pribadi, api Kristus terus bekerja dalam hati setiap orang percaya. Ini adalah api yang mengkonfrontasi dosa, membakar kebanggaan, dan mencairkan hati yang keras. Ketika kita datang kepada Kristus, kita mengalami pemurnian yang mendalam, proses yang seringkali menyakitkan tetapi esensial untuk pertumbuhan rohani. Api ini tidak hanya menghancurkan yang lama, tetapi juga membangun yang baru—karakter yang diubahkan, kasih yang tulus, dan kesetiaan yang tak tergoyahkan.

Api ini juga menuntut kita untuk menjadi agen perubahan. Jika api Kristus telah menyala dalam diri kita, maka kita tidak bisa tetap pasif. Kita terdorong untuk membagikan terang-Nya kepada dunia, untuk menjadi saksi-Nya, dan untuk membawa keadilan dan kasih-Nya ke dalam komunitas kita. Kehidupan yang dipenuhi api Kristus adalah kehidupan yang dinamis, penuh semangat, dan berorientasi pada misi.

V. Api dan Penderitaan: Sebuah Paradox Ilahi

Pernyataan Yesus tentang api ini juga segera diikuti dengan perkataan-Nya tentang baptisan yang harus Dia terima, yang secara jelas merujuk pada penderitaan dan kematian-Nya:

"Ada suatu baptisan yang harus Kuterima, dan betapa susahnya hati-Ku, sebelum semuanya itu terlaksana!" (Lukas 12:50)

Ini mengungkapkan sebuah paradox: api yang membawa pemurnian dan semangat juga erat kaitannya dengan penderitaan. Bagi Yesus, api ini adalah salib-Nya. Bagi pengikut-Nya, api ini seringkali melibatkan bagian dalam penderitaan-Nya.

1. Api Penderitaan yang Memurnikan

Alkitab sering menggambarkan penderitaan orang benar sebagai api yang memurnikan iman mereka (1 Petrus 1:6-7). Sama seperti emas diuji dalam api untuk menyingkirkan kotoran, demikian pula iman kita diuji melalui kesulitan dan kesukaran. Penderitaan, meskipun tidak menyenangkan, dapat menjadi alat yang kuat di tangan Allah untuk membentuk karakter kita, memperdalam ketergantungan kita pada-Nya, dan memperkuat iman kita.

2. Keterlibatan dalam Api Kristus

Mengikuti Yesus berarti bersedia untuk berpartisipasi dalam "api" yang Dia bawa. Ini mungkin berarti menghadapi penolakan, ejekan, atau bahkan penganiayaan karena iman kita. Ini juga berarti bersedia untuk "membakar" diri kita sendiri demi pelayanan, mengorbankan kenyamanan pribadi demi Kerajaan Allah. Api ini menuntut kita untuk melepaskan segala sesuatu yang menghalangi kita dari ketaatan penuh kepada Kristus.

Panggilan untuk mengambil salib kita sendiri (Lukas 9:23) adalah bagian intrinsik dari api ini. Ini bukan panggilan untuk penderitaan yang tidak berarti, melainkan untuk penderitaan yang memurnikan, yang menuntun pada kehidupan yang lebih dalam dan penuh makna dalam Kristus. Penderitaan dalam konteks ini adalah bagian dari api yang membakar dosa-dosa kita dan membentuk kita menjadi semakin serupa dengan-Nya.

VI. Implikasi Praktis bagi Orang Percaya Hari Ini

Bagaimana pernyataan Yesus tentang api ini relevan bagi kita yang hidup di abad ke-21?

1. Panggilan untuk Komitmen Total

Api Kristus menuntut komitmen yang tidak setengah-setengah. Ini adalah panggilan untuk menyerahkan seluruh hidup kita kepada-Nya, membiarkan api-Nya membakar habis segala sesuatu yang tidak sesuai dengan kehendak-Nya. Apakah kita bersedia membiarkan api ini bekerja dalam hati kita, bahkan jika itu berarti perubahan yang radikal dan tidak nyaman?

2. Kesiapan untuk Konflik demi Kebenaran

Jika kita menerima api ini, kita harus siap untuk perpecahan yang mungkin datang. Membela kebenaran dalam dunia yang semakin relativistik seringkali berarti berdiri sendirian, menghadapi kritik, dan mungkin kehilangan hubungan. Yesus tidak menjanjikan jalan yang mudah, tetapi jalan yang benar dan transformatif.

3. Merindukan Api Roh Kudus

Kita harus merindukan dan secara aktif mencari pencurahan Roh Kudus dalam hidup kita, baik secara pribadi maupun dalam komunitas. Seperti Yesus yang "menginginkan" api itu menyala, kita juga harus berdoa agar Roh Kudus terus mengobarkan semangat, kuasa, dan pemurnian-Nya dalam diri kita. Kita membutuhkan api ini untuk menghadapi tantangan zaman, untuk memberitakan Injil dengan berani, dan untuk hidup kudus di tengah dunia yang gelap.

4. Menjadi Pembawa Api

Sebagai pengikut Kristus, kita dipanggil untuk tidak hanya menerima api ini tetapi juga untuk menjadi pembawa api itu ke dunia. Melalui kehidupan kita, perkataan kita, dan tindakan kita, kita harus menyebarkan kebenaran, keadilan, dan kasih Kristus, yang seringkali akan menjadi api yang memisahkan, memurnikan, dan mengobarkan di mana pun kita berada. Ini adalah panggilan untuk menjadi garam dan terang, membawa perubahan yang berasal dari api ilahi.

Membawa api juga berarti kita harus memiliki keberanian untuk menentang ketidakadilan, untuk berbicara menentang dosa, dan untuk berdiri teguh pada prinsip-prinsip Alkitabiah, bahkan ketika itu tidak populer. Api Kristus memberikan keberanian untuk hidup secara kontra-kultural, untuk menjadi suara bagi yang tidak bersuara, dan untuk menjadi agen rekonsiliasi dan restorasi di tengah-tengah kehancuran.

5. Api yang Menguji Dasar Iman

Pernyataan Yesus tentang api juga berfungsi sebagai ujian fundamental bagi dasar iman kita. Apakah iman kita dibangun di atas pasir, mudah goyah oleh angin perubahan dan api pengujian? Atau apakah iman kita dibangun di atas batu karang Kristus, yang mampu bertahan bahkan ketika api melalap segala sesuatu di sekitarnya? Api ini akan mengungkapkan apa yang sejati dan apa yang palsu dalam hidup kita dan dalam Gereja.

Ini adalah pengingat bahwa hidup Kristen bukanlah perjalanan yang tanpa tantangan. Sebaliknya, itu adalah jalan yang seringkali penuh dengan kesulitan, yang dirancang oleh hikmat ilahi untuk memurnikan kita dan menjadikan kita lebih kuat. Kita tidak boleh takut pada api, karena Allah ada di dalamnya, bekerja untuk kebaikan kita.

6. Api Sebagai Sumber Kekuatan Misioner

Tanpa api Roh Kudus, upaya misioner kita akan terasa hampa. Api ini adalah yang memberikan kuasa untuk bersaksi, untuk mengajar, dan untuk melayani. Ini adalah api yang mendorong kita keluar dari zona nyaman kita dan masuk ke dalam dunia untuk membagikan Kabar Baik. Kerinduan Yesus agar api itu menyala adalah kerinduan agar misi Kerajaan Allah dimanifestasikan secara penuh di bumi, mengubah hati dan masyarakat. Kita dipanggil untuk menjadi bahan bakar bagi api ini, memungkinkan Roh Kudus untuk bekerja melalui kita dengan cara yang kuat.

VII. Mengapa Begitu Penting Bahwa "Api Itu Sudah Menyala"?

Ungkapan "alangkah Aku menginginkan jika api itu sudah menyala!" menunjukkan urgensi yang luar biasa. Mengapa Yesus begitu mendesak agar api ini segera berkobar?

1. Kesegeraan Transformasi

Yesus tahu bahwa waktu terbatas. Dunia sangat membutuhkan pemurnian, pencerahan, dan semangat yang dibawa oleh api ini. Penundaan berarti lebih banyak jiwa yang tersesat, lebih banyak penderitaan karena dosa, dan lebih banyak kesempatan yang terbuang. Keinginan-Nya adalah agar transformasi segera dimulai, agar Kerajaan Allah segera datang dengan kuasa penuh.

2. Pentingnya Penggenapan Rencana Ilahi

Pencurahan Roh Kudus adalah titik balik krusial dalam rencana penyelamatan Allah. Tanpa api ini, misi Gereja tidak akan pernah bisa dimulai dengan kekuatan yang dibutuhkan. Yesus merindukan penggenapan janji Bapa, dimulainya era Roh Kudus yang baru, yang akan memberdayakan para pengikut-Nya untuk melanjutkan pekerjaan-Nya di bumi.

3. Kedalaman Kasih Ilahi

Keinginan Yesus agar api itu menyala bukan hanya tentang pemenuhan nubuat, tetapi juga tentang kasih-Nya yang mendalam bagi umat manusia. Dia tahu bahwa api ini adalah satu-satunya jalan menuju kebebasan sejati, pemurnian, dan kehidupan yang berlimpah bagi mereka yang akan menerima-Nya. Ini adalah kasih yang rela melihat perpecahan dan penderitaan sementara demi kebaikan abadi.

Ayat Lukas 12:49 adalah pengingat yang kuat bahwa misi Yesus lebih dari sekadar menawarkan kedamaian yang mudah. Ini adalah misi yang radikal, transformatif, dan seringkali menantang, di mana api ilahi membakar untuk memurnikan, memisahkan, dan mengobarkan. Ini adalah panggilan untuk setiap orang percaya untuk merenungkan: Apakah api ini menyala dalam hati saya? Apakah saya bersedia menerima dampaknya, bahkan jika itu berarti menghadapi perpecahan dan penderitaan? Dan apakah saya siap untuk menjadi pembawa api ini, menyebarkannya ke seluruh bumi hingga keinginan Kristus terpenuhi?

Dalam dunia yang seringkali mencari kenyamanan dan menghindari konflik, pesan dari Lukas 12:49 adalah tantangan yang relevan. Ini adalah panggilan untuk keberanian rohani, untuk otentisitas iman, dan untuk gairah yang tak tergoyahkan bagi Kerajaan Allah. Marilah kita merindukan, seperti Yesus, agar api itu benar-benar menyala, bukan hanya di dalam diri kita, tetapi juga di seluruh bumi, membawa pembaruan, kebenaran, dan semangat yang diidam-idamkan Kristus.

Memahami Lukas 12:49 secara mendalam membawa kita pada pemahaman yang lebih kaya tentang kompleksitas dan kekayaan misi Yesus Kristus. Ini bukan gambaran Mesias yang pasif atau sekadar penghibur, melainkan seorang Raja yang datang dengan kuasa dan tujuan yang membara. Api yang Dia bawa adalah api yang tak terhindarkan dan tak terhentikan, yang akan terus bekerja hingga Kerajaan Allah ditegakkan sepenuhnya, membersihkan segala sesuatu yang tidak berkenan di hadapan-Nya, dan mengobarkan setiap hati yang bersedia untuk menerima-Nya.

Oleh karena itu, marilah kita, sebagai pengikut-Nya, menerima api ini dengan kerendahan hati dan keberanian. Biarkan api-Nya membakar dalam diri kita, memurnikan kita, mengobarkan semangat kita, dan memampukan kita untuk menjadi agen perubahan yang sejati di dunia ini. Karena hanya dengan api yang menyala inilah kita dapat benar-benar memenuhi panggilan kita untuk menjadi terang dan garam bumi, membawa harapan dan transformasi ilahi kepada setiap sudut kehidupan.
